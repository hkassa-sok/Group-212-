# Gin Rummy 
## Project Description 
This project is a text based version of the card game Gin Rummmy using python and OOP. This game allows two players to 
take turns drawing and discarding cards while attempting to match sets and reduce their deadwood points.
This project highlights the usage of all the skills we learned in 326, like classes, methods, loops and much more.

# Files 
## main.py 
Starts the game 
## game.py 
The main game loop, tutorial system, and the gameplay logic
## player.py 
Define the Player class, and manages the sets, card discarding and deadwood calculations
## deck.py 
Creates and shuffles the deck of cards and allows cards to be drawn
## card.py 
Defines individual card object including rand and point value. 

# How to run program 
python main.py 

## How to play
Each player starts with 10 cards 
1. Player draws a card from the deck
2. Player discards one card from their hand
3. The game checks for sets and calculates deadwood points
